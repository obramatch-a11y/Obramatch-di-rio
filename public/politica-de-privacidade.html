Política de Privacidade

ObraMatch Diário — registro diário de execução de obras

Última atualização: 08 de julho de 2026.

1. Quem somos

O aplicativo ObraMatch Diário é operado pela ObraMatch (obramatch.com.br), plataforma brasileira de serviços para o setor de construção civil, atualmente operada por pessoa física responsável. Contato: suporte@obramatch.com.br.

2. Dados que coletamos

  • E-mail e senha de acesso, para autenticação, por meio do serviço Google Firebase Authentication. A senha é criptografada e não é visível para nós.
  • Nome ou identificação informados no cadastro.
  • Dados das obras e relatórios diários (RDOs): descrições, atividades, efetivo, equipamentos e ocorrências registradas pelo usuário, armazenados no banco de dados Cloud Firestore (Google).
  • Fotos das obras enviadas pelo usuário, armazenadas no serviço Supabase Storage.
  • Localização GPS aproximada (opcional), usada exclusivamente para preencher automaticamente as condições climáticas do relatório, por meio do serviço Open-Meteo. O usuário pode negar a permissão de localização e preencher o clima manualmente.
  • Áudios e fotos enviados voluntariamente pelo bot do Telegram, processados por inteligência artificial (Google Gemini) exclusivamente para preencher o relatório diário do próprio usuário.
  • Assinaturas digitais desenhadas no aplicativo para validação dos relatórios.

3. Para que usamos os dados

Os dados são usados exclusivamente para o funcionamento do aplicativo: criar, armazenar e gerar em PDF os relatórios diários de obra do próprio usuário. Cada relatório recebe um código de integridade (hash SHA-256) para comprovar que não foi alterado após a emissão.

4. Com quem compartilhamos

Não vendemos dados pessoais. Não exibimos anúncios. Não usamos rastreadores de publicidade. Os dados são compartilhados apenas com os provedores de infraestrutura estritamente necessários ao funcionamento do serviço: Google Firebase (autenticação e banco de dados), Supabase (armazenamento de fotos), Cloudflare (hospedagem), Open-Meteo (clima, quando o usuário permite localização), Telegram e Google Gemini (apenas quando o usuário utiliza o bot do Telegram).

5. Armazenamento e segurança

Todos os dados trafegam por conexão segura (HTTPS). O acesso aos dados é restrito à conta do próprio usuário. Os relatórios recebem hash de integridade SHA-256.

6. Seus direitos (LGPD)

Nos termos da Lei Geral de Proteção de Dados (Lei nº 13.709/2018), você pode solicitar a qualquer momento o acesso, a correção ou a exclusão dos seus dados. Para excluir sua conta e todos os dados associados, siga as instruções da página Exclusão de conta e dados.

7. Retenção dos dados

Os dados são mantidos enquanto a conta do usuário existir. Após a solicitação de exclusão, os dados são removidos, ressalvados registros mínimos cuja guarda seja exigida por lei, pelo prazo legal.

8. Menores de idade

O ObraMatch Diário é uma ferramenta de uso profissional destinada a maiores de 18 anos. Não coletamos intencionalmente dados de menores.

9. Alterações desta política

Esta política pode ser atualizada periodicamente. A data da última atualização consta no topo desta página. Alterações relevantes serão comunicadas no aplicativo.

10. Contato

Dúvidas sobre privacidade e proteção de dados: suporte@obramatch.com.br
